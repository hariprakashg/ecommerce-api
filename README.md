# E-Commerce API Backend
A RESTful API built with Python, Flask, and MongoDB Atlas.

## Features
* **JWT Authentication**: Secure login and registration.
* **Product Management**: CRUD operations for e-commerce products.
* **Database**: Hosted on MongoDB Atlas with optimized indexing.

## Setup
1. Clone the repo: `git clone <your-repo-url>`
2. Create a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file and add your `MONGO_URI` and `JWT_SECRET`.
5. Run the app: `python run.py`
